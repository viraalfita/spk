# SPK Creator

Digital work order and payment tracking system built with Next.js 14 + Supabase.

## Quick Start

```bash
# 1. Run setup script
./setup.sh

# 2. Configure Supabase in .env.local

# 3. Run database migration (see README.md)

# 4. Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

✅ Admin dashboard for SPK management  
✅ Real-time payment tracking  
✅ Vendor read-only portal  
✅ PDF generation  
✅ n8n webhook integration  
✅ Full TypeScript & validation
